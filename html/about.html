El sitio web "Digital GANP" ha sido desarrollada a través de HTML, Javascript y CSS. El lenguaje de etiquetas HyperText Markup Language o HTML se empleó para estructurar el contenido de la página, Cascading Style Sheets o CSS sirvió para construir el estilo visual, y el lenguaje de programación JavaScript para desarrollar los elementos interactivos, cómo, animaciones y direccionamiento de enlaces. Se utilizó el repositorio de GitHub para mantener Digital GANP en línea, utilizando su servicio de GitHub Pages como servidor en nube, acompañado del dominio www.digitalganp.com, que ha sido registrado por medio de Hostinger. Agradecimiento especial a Manuelalejandro Buitrago, por su excepcional trabajo como programador principal encargado del desarrollo de la página web.
